# Project Roadmap

## Phase 1 — General Architecture

Define the technical architecture, repository structure, testing strategy and Git workflow.

## Phase 2 — Spring Boot Backend

Create the backend API using Java, Spring Boot and Maven.

## Phase 3 — MySQL Database

Integrate the backend with a relational MySQL database.

## Phase 4 — Swagger / OpenAPI

Document the API using Swagger and OpenAPI.

## Phase 5 — JWT Authentication

Implement authentication and authorization using Spring Security and JWT.

## Phase 6 — Docker

Containerize the backend and database.

## Phase 7 — Postman

Create Postman collections and environments.

## Phase 8 — Cypress Framework

Build a Cypress automation framework.

## Phase 9 — API Testing

Automate API validations.

## Phase 10 — Mocking and Intercept

Use Cypress intercepts to mock network behavior.

## Phase 11 — Selenium + TestNG

Build a Java-based UI automation framework.

## Phase 12 — REST Assured

Create Java-based API automation tests.

## Phase 13 — GitHub Actions

Create CI pipelines.

## Phase 14 — Azure DevOps

Add Azure DevOps pipeline support.

## Phase 15 — Professional README

Create final technical project documentation.

## Phase 16 — Deploy

Deploy the application and execute tests against a live environment.
